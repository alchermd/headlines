Technology

The core application is built with the Python microframework Flask and the front end with Bootstrap. The data is pulled using the requests and urllib libraries. The news, weather, and forex data are fetched from the following APIs:

NewsAPI OpenWeatherMap OpenExchangeRates

More Information

This application is inspired by the headlines project from the book Flask by Example. The application is hosted in Python Anywhere, in which they are generous enough to provide a free tier of their services. With the limitations imposed by the free tier account, the RSS feeds used in the book was replaced with external APIs that are fetched through the requests and urllib libraries.

Note that this application is under continuous development. That means that features and fixes are added from time to time, so be sure to check back for updates. Or better yet, contribute your thoughts at the project's Github Repository.
